# RKM Ledger (Khata App)

A modern, responsive web application for managing customer ledgers, credits, and payments.

## Features
- **Dashboard**: View total outstanding amount and recent activity.
- **Customer Management**: Add and search customers by name or phone.
- **Ledger**: Record 'Given Goods' (Credit) and 'Got Payment' (Debit). Automatically calculates balance.
- **WhatsApp Integration**: Share payment reminders with one click.
- **Data Persistence**: Data is saved locally on your device (LocalStorage).
- **Export**: Export customer list to CSV.

## Tech Stack
- Frontend: React + Vite
- Styling: Vanilla CSS (Custom Design System)
- Icons: Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
